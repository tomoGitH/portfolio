# Features

３Dアクションゲーム

# Requirement

Windows

# Usage

・ゲームのプレイ

「Win.zip」を解凍 -> 「Win/Crazy Plant.exe」を実行

# Controller

・ゲームパッド
Lスティック - 移動
Rスティック - カメラ回転
R1 + Lスティックを押した状態でR1を離す - アタック

・キーボード
WASD - 移動
マウス - カメラ回転
スペース + WASDを押した状態でR1を離す - アタック
